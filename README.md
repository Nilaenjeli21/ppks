# Nodejs / Expressjs API

## Commands
Init Project
```
yarn
```

Since this project use Prisma need run this before start service
```
npx prisma generate
npx prisma migrate dev
```

Start Development Mode
```
yarn watch
```

Build Project
```
yarn Build
```

Start Project
```
yarn start
```
